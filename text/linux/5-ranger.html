ranger配置文件

~/.config/ranger/rc.conf #文件地址

写一个自己ranger

ranger --copy-config=all #这样就会在上面地址生成ranger的配置文件，但是它会多一句话，如果你完全想用自己的配置文件，需要在环境变量加一句话

export RANGER_LOAD_DEFAULT_RC FALSE #在bash下用这个增加环境变量
set -g -x RANGER_LOAD_DEFAULT_RC FALSE #在fish下用这个设置环境变量
如果想要每次启动都自动帮忙配置好环境变量
在~/.config/fish/config.fish 下增加上面再fish下设置环境变量的话
