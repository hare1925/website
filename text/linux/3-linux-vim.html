vim分屏

打开vimrc文件
按下shift+上下左右，国标移动会快一些，5行5行的移动。
普通模式下：
split 上下分屏，光标在上面
vsplit 左右分屏，光标会放在左边

那国标怎么才能放到右边呢？
:set splitright
:vsplit
执行以上两个就好了

上面这样写实在太麻烦了，只好map一下啊。
map si :set splitright:vsplit #这个就往右边分屏，光标在右边
map sn :set nosplitright:vsplit #这个就往左边分屏，光标在左边
map su :set nosplitbelow:vsplit #这个就往上边分屏，光标在上边
map se :set nosplitbelow:vsplit #这个就往下边分屏，光标在下边
